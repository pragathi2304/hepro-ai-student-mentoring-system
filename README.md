# HEPro AI+ Dedicated Mentoring System

## Overview

HEPro AI+ is an AI-driven student mentoring intelligence platform designed to identify at-risk students, recommend suitable mentors, and generate personalized intervention plans. The system combines rule-based scoring with machine learning techniques to improve student support and mentoring outcomes.

## Features

* Student Readiness Index (SRI)
* Academic, Wellness, Productivity, and Career Readiness Scoring
* K-Means Clustering for Student Segmentation
* Mentor Recommendation Engine
* Risk Detection and Alert Generation
* Personalized Intervention Strategies

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Machine Learning Workflow

1. Data Collection and Preprocessing
2. Feature Engineering
3. Rule-Based Scoring
4. Student Segmentation using K-Means Clustering
5. Mentor Matching
6. Intervention Recommendation

## Results

* Identified At-Risk Students
* Classified High Performers and Career-Confused Students
* Automated Mentor Allocation
* Generated Personalized Intervention Plans
